# Antigravity Agent Rules

The following rules apply specifically to the Disha project:

## Documentation Syncing
Whenever you create or update planning artifacts (such as `implementation_plan.md`, `task.md`, or `walkthrough.md`) in your private brain directory, you MUST also sync a copy of them to the `docs/antigravity/` directory in the project repository. 
This allows human developers (and other tools like Opencode) to manually track your plans and progress.

**Sync Paths:**
- `implementation_plan.md` -> `docs/antigravity/implementation_plan.md`
- `task.md` -> `docs/antigravity/tasks.md`
- `walkthrough.md` -> `docs/antigravity/walkthrough.md`
